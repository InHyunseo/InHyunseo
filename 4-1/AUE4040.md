## 12. NumPy로 kNN 직접 구현

```python
class KNearestNeighbor_Numpy:
    def fit(self, X, y):
        self.X_train = np.array(X)
        self.y_train = np.array(y)

    def compute_distances(self, X_test):
        X_test = np.array(X_test)

        test_sq = np.sum(np.square(X_test), axis=1, keepdims=True)
        train_sq = np.sum(np.square(self.X_train), axis=1)
        cross_term = np.dot(X_test, self.X_train.T)

        dists = np.sqrt(np.maximum(test_sq - 2 * cross_term + train_sq, 0.0))
        return dists

    def predict(self, X_test, k=1):
        dists = self.compute_distances(X_test)
        num_test = dists.shape[0]
        y_pred = np.zeros(num_test)

        for i in range(num_test):
            closest_idx = np.argsort(dists[i, :])[:k]
            closest_y = self.y_train[closest_idx]
            counts = np.bincount(closest_y.astype(int))
            y_pred[i] = np.argmax(counts)

        return y_pred
```

### `fit`

원리:

- kNN은 lazy learning이라 학습 데이터를 저장하는 것이 핵심이다.

특징:

- 신경망처럼 가중치를 업데이트하지 않는다.

### `compute_distances`

핵심 식:

`||a-b||^2 = ||a||^2 - 2a·b + ||b||^2`

함수 사용법:

- `np.square(X)`
- `np.sum(..., axis=1, keepdims=True)`
- `np.dot(A, B.T)`
- `np.maximum(x, 0.0)`
- `np.sqrt(x)`

원리:

- 테스트 샘플과 학습 샘플 간 모든 쌍의 거리를 한 번에 계산한다.

특징:

- 반복문 대신 브로드캐스팅과 행렬곱을 이용해 빠르게 계산한다.

### `predict`

함수 사용법:

- `np.argsort(arr)[:k]`
- `np.bincount(labels.astype(int))`
- `np.argmax(counts)`

원리:

- 가장 가까운 `k`개를 고른 뒤 다수결로 클래스를 정한다.

특징:

- kNN의 핵심 원리를 그대로 보여주는 구현이다.
